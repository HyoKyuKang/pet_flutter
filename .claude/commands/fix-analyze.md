# /fix-analyze

`flutter analyze` 오류를 모두 찾아 수정한다.

## 절차
1. `flutter analyze` 실행해서 오류 목록 확인
2. 오류가 없으면 "이미 통과 상태입니다" 출력 후 종료
3. 오류가 있으면 Explore 에이전트로 해당 파일 분석
4. 각 오류를 순서대로 수정한다
5. 수정 후 다시 `flutter analyze` 실행해서 통과 확인
6. 통과할 때까지 반복한다

## 주의
- 오류 수정 시 기존 로직 변경 금지, 오류만 수정
- 경고(warning)도 함께 수정
- 수정한 파일 목록과 내용을 마지막에 요약
